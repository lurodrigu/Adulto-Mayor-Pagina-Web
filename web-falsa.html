CÓMO IDENTIFICAR UNA WEB FALSA

1. Verifica el dominio principal

Asegúrate de que el dominio sea legítimo y exacto.

Ejemplo real: www.paypal.com

Falso: www.paypa1.com, paypal.security-check.com, paypal.com.login.update.ru

El dominio principal está justo antes del .com, .net, etc.

2. Revisa el protocolo HTTPS

Asegúrate de que la URL comience con https://, no solo http://.

Busca el ícono del candado en el navegador.

Nota: Esto no garantiza que el sitio sea seguro, pero sí es una señal inicial.

3. Evalúa la estructura y gramática

Las URLs falsas suelen tener:

  • Palabras mal escritas (errores ortográficos)
  • Nombres largos o con muchos guiones
  • Texto confuso o traducciones automáticas

4. No hagas clic en enlaces sospechosos

Si recibes un enlace por correo, mensaje o redes sociales, no hagas clic inmediatamente.

Pasa el cursor sobre el enlace para ver la URL completa y verifica que te lleve al sitio correcto antes de hacer clic.

5. Consulta con herramientas de verificación

Usa servicios como:

  • VirusTotal
  • Google Safe Browsing

Pega la URL allí y verifica si es considerada peligrosa.

6. Comprueba la información del dominio

Visita Whois Lookup

Revisa quién registró el dominio, cuándo fue creado y si parece confiable.

7. Desconfía de urgencias o amenazas

Los sitios falsos suelen decir cosas como:

  • “Tu cuenta será suspendida”
  • “Actualiza tu información AHORA”
  • “Gana un premio urgente”

Los sitios legítimos rara vez usan ese tipo de lenguaje.
